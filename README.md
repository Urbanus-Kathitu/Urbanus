The projects in this repository were developed using Python 3 and R. The following Python packages are required to run the Python scripts:

Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
The following R packages are required to run the R scripts:

tidyverse
dplyr
ggplot2
caret
